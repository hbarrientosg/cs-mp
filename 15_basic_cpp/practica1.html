Prácticas de Estructuras de Datos II Práctica 1: Clases, constructores y sobrecarga de operadores
(valoración 10%)

Objetivos:

El objetivo de esta práctica es que el alumno de Estructura de Datos II practique el uso de clases, constructores y sobrecarga de operadores en C++.

Prerrequisitos:

Haber estudiado las clases en C++.

Se pide

    Cree una clase denominada "Fraccion" que represente un número fraccionario y que tenga como funciones miembro las siguientes:

  • Constructor: Añada el constructor o los constructores necesarios para crear una fracción pasándole el numerador y el denominador (en este orden), que permita conversiones automáticas de entero a Fraccion y que permita crear vectores de objetos Fraccion inicializados a la fracción 0/1.
  • numerador: Devuelve el numerador de la fracción.
  • denominador: Devuelve el denominador de la fracción.
  • valor: Devuelve el valor real (float) de la fracción. En caso de que el denominador sea cero, el resultado de esta operación será FLT_MAX. Para usar FLT_MAX hay que incluir <cfloat> y usar el espacio de nombres std.
  • Operador -: Función que devuelve una fracción que sea resultado de la resta de la fracción actual menos una pasada por parámetro.
  • Operador /: Análogo a la resta.
  • inversa: Función que devuelve la fracción inversa de la actual.
  • numeroMixto: Para una fracción A/B el número mixto se define como la terna (C,D,B) donde A/B=C+D/B siendo C la parte entera del cociente entre A y B, y D el resto. Escriba un procedimiento que devuelva en tres parámetros C, D y B, el número mixto de la fracción. En caso de que B sea cero se debe devolver INT_MAX como C y D, y B igual a cero. Para usar INT_MAX hay que incluir <climits> y usar el espacio de nombres std
  • elevadoA: Escriba una función que pasándole un entero devuelva una fracción que sea el resultado de elevar la fracción a dicho entero.
  • Añada las funciones miembro que considere oportunas para alcanzar los objetivos de funcionamiento. No añada funciones o procedimientos que no sean para apoyar las solicitadas.

Además, escriba las siguientes funciones no miembros:

  • Operador +: Análogo a la resta pero una función no perteneciente a la clase.
  • Operador *: Análogo a la resta pero una función no perteneciente a la clase.

Todas las operaciones anteriores deben poderse realizar con objetos constantes.

Exceptuando el constructor o constructores ninguna función puede modificar una fracción.

Las fracciones deben normalizarse en el momento de su inicialización. Una fracción se dice que está normalizada si:

  1. El numerador y el denominador no tienen factores comunes.
  2. El denominador siempre es positivo
  3. Si la fracción representa un valor negativo, el signo lo tiene el numerador.

Para calcular el máximo común divisor se puede emplear el algoritmo de Euclides que se muestra a continuación:

Función Entero Euclides(m,n)
variables entrada entero m,n
          local entero t
{Precondición: n >= 1 ^ m >= 0}
mientras m >0 hacer
    t <- n mod m {devuelve el resto de la división entre n y m} 
    n <- m
    m <- t
fin mientras
devolver n

Para calcular el valor absoluto se puede emplear la función abs(int), declarada en <cstdlib>, y que nos devuelve el valor absoluto del entero pasado.

Escriba la clase Fraccion en los ficheros fraccion.h y fraccion.cpp. Implemente todas las funciones y/o procedimientos en fraccion.cpp, en el fichero fraccion.h no se debe implementar ninguna función.

Ejemplo de uso de Fraccion

#include "fraccion.h"
int main()
{
    const int NV=10;
    Fraccion a;
    Fraccion vector[NV];
    const Fraccion b(2,3), c(-1);
    a=b-c;
    a=a+b/3;
    for(int i=1; i< NV; i++) vector[i]=vector[i-1]+b+a;
    return 0;
}

Pruebas a realizar

Escriba un programa principal en un fichero denominado pruebafraccion.cpp que compruebe la funcionalidad de cada una de las operaciones de Fraccion.
