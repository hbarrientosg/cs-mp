Prácticas de Estructuras de Datos II Práctica 2: Operador de asignación y constructor de copia.
(Valoración 20%)

Objetivos:

El objetivo de la práctica es que el alumno practique el uso del operador de asignación y el constructor de copia en C++ y la construcción de clases complejas.

Ejercicio:

Cree una clase Polinomio que emule los polinomios con coeficientes reales.
(anxn+ an-1xn-1+ ...+a2x2+ a1x1+ a0x0)

Esta clase debe tener implementadas las siguientes funciones/operaciones:

  • Un constructor al que se le pase el grado del polinomio a almacenar. El constructor debe inicializar los coeficientes del polinomio a cero.
  • operator [ ]. Pasándole un entero i devuelve una referencia al coeficiente ai. Este operador será el único que nos permita modificar el valor de cada coeficiente.
  • grado. Una función que devuelva el grado del polinomio.
  • muestra. Un procedimiento que muestra el polinomio con cout. No muestre los términos con coeficiente cero. En caso de que todos los coeficientes sean cero, muestra debe mostrar un cero.
  • valor. Una función que pasándole un valor real  "v", nos devuelva el valor del polinomio para x=v.
  • Las que se consideren necesarias para el correcto funcionamiento de la clase en circunstancias normales, como poder asignar un polinomio a otro, pasarlo por valor a una función, etc.

También cree las siguientes funciones no miembro de la clase pero relacionadas

  • operator +. Una función que devuelve el polinomio suma de dos polinomios.
  • operator -. Una función que devuelve el polinomio resta de dos polinomios.
  • operator -. Una función que devuelve la negación de un polinomio.
  • operator *. Una función que devuelve el polinomio multiplicación de dos polinomios.
Por simplicidad en la gestión del grado se debe considerar que el polinomio siempre tiene el grado original, aunque los coeficientes de mayor grado se vuelvan cero. Esto implica que :
  • La función grado devuelve el grado con el que se crea el polinomio.
  • Las funciones suma y resta devuelven un polinomio de grado el mayor de los dos operandos.
  • La función negación devuelve un polinomio del mismo grado que el operando.
  • La función multiplicación devuelve un polinomio de grado la suma de los grados de los operandos.

Implementación:

Para implementar la clase Polinomio utilice un vector de reales (float) creado en memoria dinámica. Suponga que el elemento i del vector representa ai.

Escriba la clase Polinomio en los ficheros polinomio.h y polinomio.cpp. Implemente todas las funciones y/o procedimientos en polinomio.cpp (en el fichero polinomio.h no se debe implementar ninguna función o procedimiento)

Pruebas de ejecución

Escriba un programa principal, en un fichero llamado pruebapolinomio.cpp, que compruebe la funcionalidad de todas las características solicitadas para la clase Polinomio.

Ejemplo de uso de la clase Polinomio

#include <iostream>
using namespace std;
#include "polinomio.h"
int main()
{
    Polinomio p(3);
    p[0]=1;
    p[1]=-1;
    p[2]=2;
    p[3]=-2;
    p.muestra();
    for(float i=-3; i<=3; i+=0.5)
      cout << "Para "<<i << " da "<< p.valor(i)<< endl;
    return 0;
}

Entrega de prácticas

Se debe entregar el fichero polinomio.h, polinomio.cpp y pruebapolinomio.cpp.
